Projeto SETEC: Chatbot com Inteligência Artificial

Resumo

O chatbot inteligente proposto usa Inteligência Artificial para responder automaticamente perguntas feitas por usuários sobre a escola. Ele pode ser integrado com plataformas de chat como Telegram ou em uma página web.

Objetivos

  • Oferecer atendimento automatizado aos alunos
  • Melhorar a experiência dos visitantes da escola
  • Demonstrar aplicação prática de IA e NLP

Tecnologias

  • Python (Flask/FastAPI)
  • API do ChatGPT
  • Telegram Bot API
  • HTML + CSS + JS para interface web
