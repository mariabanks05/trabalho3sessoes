Projeto da SETEC: Chatbot com Inteligência Artificial

Sobre o Projeto

Este chatbot visa automatizar o atendimento escolar, podendo ser utilizado por alunos, visitantes e professores. A ideia é simular uma conversa natural com o usuário usando uma IA.

Etapas do Projeto

  • 1. Definição de escopo e fluxos de conversa
  • 2. Treinamento com perguntas frequentes
  • 3. Conexão com API da OpenAI
  • 4. Implementação Web ou via Telegram

Ferramentas e Linguagens

  • Python (Flask ou FastAPI)
  • JavaScript para interações
  • API do ChatGPT (OpenAI)
  • HTML/CSS para interface
