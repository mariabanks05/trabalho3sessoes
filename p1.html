Projeto SETEC: Chatbot Inteligente

Objetivo do Projeto

Desenvolver um chatbot utilizando Inteligência Artificial para atender dúvidas frequentes dos alunos, professores e visitantes da escola.

Tecnologias Utilizadas

  • Linguagem Python
  • API do ChatGPT (OpenAI)
  • Plataformas de bot como Telegram ou WhatsApp
  • HTML/CSS para página de integração

Funcionamento

O chatbot interpreta as perguntas feitas pelos usuários usando NLP (Processamento de Linguagem Natural) e responde com base em dados pré-definidos ou em conexão com uma IA.
